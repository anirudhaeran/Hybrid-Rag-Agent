How it Works

1

AI Agent Decision

AI agent intelligently determines whether to use GraphRAG or SQL based on the query type and data source

2

Query Processing

Queries are sent to LightRAG server for knowledge graph processing, or SQL commands are generated for database queries

3

Answer Generation

LightRAG answers questions via knowledge graph, or SQL queries retrieve data from Postgres database

4

Response Delivery

Answers are returned to the AI Agent and presented to you in a clear, conversational format
